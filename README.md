# Controls

## Player
- W A S D -> Movement
- Mouse Movement -> Look
- Space -> Jump
- Left Click -> Shoot
- G -> Spawn Next Wave
- F -> Use Class Ability

## Tower
- Right Click -> Place Tower where you are looking
- B -> Open Buy Menu

## Misc
- ESC -> View Pause Menu (and controls)
