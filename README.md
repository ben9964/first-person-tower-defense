# Controls

- W A S D -> Movement
- Mouse Movement -> Look
- Space -> Jump
- Left Click -> Shoot
- Right Click -> Place Tower where you are looking
- G -> Spawn Next Wave
- ESC -> View Pause Menu (and controls)
